# events_frontend_app

Next.js (App Router) frontend for the Event Hub Platform.

## Setup

1) Copy env template:

```bash
cp .env.example .env.local
```

2) Set:

- `NEXT_PUBLIC_API_BASE_URL` (FastAPI base URL)
- `NEXT_PUBLIC_SITE_URL` (this app's base URL)

3) Install and run:

```bash
npm i
npm run dev
```
